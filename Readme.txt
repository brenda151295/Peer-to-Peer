Compile:
	make all
Clean:
	make clean
Run:
	./server
	./client

Interaction:

1. The user have to sign in the tracker, using his/her IP and Port. In this case we use the local IP "127.0.0.1". And the port of the server is 9000.

2. The user has two options; first, list of the peers and upload a file.

3. Upload a file: The user have to enter the name of the file, in this case is "arbol.bin"

4. Download a file: The user have to enter the name of the file, in this case is "arbol.bin" then you can check your files an found a file named "arbol.bin"
